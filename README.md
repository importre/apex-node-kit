# ∆pex-node-kit

Riiid flavored [apex](https://github.com/apex/apex) starter kit.
Write your own lambda function with `webpack` & `babel`. Enjoy :alien:

## start

```
$ git clone riiid/∆pex-node-kit <YOUR_DIR>
$ cd <YOUR_DIR>
$ npm run new
```

update project.json (`name`, `role`) ...

## deploy function

```
$ apex deploy
$ apex invoke hello
$ apex logs hello
```

## deploy api gateway

```
$ apex infra get
$ apex infra plan
$ apex infra apply
```

## development

for lint,

```
$ npm run lint
```
