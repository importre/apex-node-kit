# ∆pex-node-kit

Riiid flavored [apex](https://github.com/apex/apex) starter kit.
Write your own lambda function with `webpack` & `babel`. Enjoy :alien:

## start

```
$ git clone riiid/∆pex-node-kit <YOUR_DIR>
$ cd <YOUR_DIR>
$ npm run new
$ apex init
$ apex deploy
$ apex invoke hello
```

## development

for lint,

```
$ npm run lint
```
